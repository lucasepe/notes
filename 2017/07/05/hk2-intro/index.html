Posted in HK2, Java

HK2 - Hundred-Kilobyte Kernel

A light-weight and dynamic dependency injection framework, an implementation of JSR-330.

HK2 also has powerful features that can be used to perform tasks such as looking up services or customizing you injections.

Some of the features of HK2 DI Kernel are

  • Custom scopes
  • Use of proxies
  • Custom injection resolution
  • Assisted injection
  • Just In Time injection resolution

Minimum Maven Dependency

<dependency>
  <groupId>org.glassfish.hk2</groupId>
  <artifactId>hk2-locator</artifactId>
  <version>2.5.0-b36</version>
</dependency>

The hk2-locator project contains the implementation of the hk2 API.

So, the ability to automatically detect services is not available.

All HK2 objects must be added programmatically and gotten with the HK2 API.
