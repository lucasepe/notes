Install Maven on Linux

Posted in Maven, Java, Linux

Install Maven on Linux

Download the latest version from the Apache Maven site: http://maven.apache.org/download.cgi. Choose the binary tar.gz file; example: (apache-maven-3.5.0-bin.tar.gz)

Extract the tar.gz archive into the /opt/apache/ directory:

sudo mkdir /opt/apache/
sudo tar xvzf ~/Downloads/apache-maven-3.5.0-bin.tar.gz -C /opt/apache/

Since variable expansion does not work in /etc/environment, create the file /etc/profile.d/maven.sh and add the following:

export M2_HOME='/opt/apache/apache-maven-3.5.0'
export M2="$M2_HOME/bin"
export MAVEN_OPTS='-Xms256m -Xmx512m'
export PATH="$PATH:$M2"

Files with the .sh extension in the /etc/profile.d directory get executed whenever a bash login shell is entered (e.g. when logging in from the console or over ssh), as well as by the DisplayManager when the desktop session loads.

To check that everything goes fine, logout, login again, open a terminal and in the shell type: mvn --version. The output shoulbe something similiar to this:
