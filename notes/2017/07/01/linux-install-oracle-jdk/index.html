How to install Oracle JDK on Linux

Download Oracle Java SE JDK8 for Linux from here

http://www.oracle.com/technetwork/java/javase/downloads/index.html

Extract the tar.gz archive into the /opt/oracle/ directory:

sudo mkdir /opt/oracle
sudo tar xvzf ~/Downloads/jdk-8*.tar.gz -C /opt/oracle/

Set Oracle JDK as the default JVM

update-alternatives --install /usr/bin/java java /opt/oracle/jdk1.8.0_141/bin/java 100
update-alternatives --install /usr/bin/javac javac /opt/oracle/jdk1.8.0_141/bin/javac 100

Verify that java has been successfully configured by running:

update-alternatives --display java
update-alternatives --display javac
